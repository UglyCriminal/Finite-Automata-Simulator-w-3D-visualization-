## 📌 About the Project

This project is an **interactive web-based simulator** for designing and visualizing:

* Deterministic Finite Automata (**DFA**)
* Non-Deterministic Finite Automata (**NFA**)

It allows users to create states, define transitions, and simulate input strings with real-time visualization in both **2D and 3D environments**.

---

## ✨ Features

✔ Add and manage states
✔ Define transitions between states
✔ Set Start and Accept states
✔ Step-by-step simulation
✔ Live state highlighting
✔ 2D graph visualization (Cytoscape.js)
✔ 3D visualization (Three.js)
✔ Simulation log tracking
✔ Interactive UI
